DOCUMENTATION

OVERVIEW

This template wants to provide a simple way to build quickly little site, with several html static pages.

It's created to be used with 3 main languages (french - default language, english and spanich).

USAGE

Download template and copy it to your local server or your ftp location.

Cloner le dépot

Comment cloner le dépot avec Aptana?

  • Aptana
  • Télécharger le plugin aptana pour git, en utilisant le menu Help>Install Aptana Features...
  • Une fois le plugin installé, utiliser le menu Window>Open Perspective, et ouvrez la perspective git.
  • Dans cette perspective, cliquer sur l'icône Clone, sous l'explorateur de projets.
  • Entrer le chemin donner plus haut, git@github.com:JeanPerriault/Bilazea-template.git
  • Cloner le dépôt, puis faites un click droit, choisissez Import Projects...
  • Depending on your server location and your aptana workspace.
  • In my case, my workspace is also my server root, so I just have to:
    • Choose import as General project, keeping the same directory name.
      • Go to your broswer, and it’s up: yes I see my template at this address: http:/localhost/Bilazea-template
