ERROR 403

Description de l'erreur:

L’erreur 403 est un code d’erreur dans le protocole HTTP. Ce code est renvoyé lorsque l’authentification est refusée. Contrairement à l’erreur 401, aucune demande d’authentification ne sera faite.
